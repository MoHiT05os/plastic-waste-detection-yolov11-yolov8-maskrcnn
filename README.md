# Plastic Waste Detection using YOLOv11, YOLOv8 and Mask R-CNN

This repository contains the implementation, training configuration, and experimental setup used in the research paper:

**"Comparative Analysis of YOLOv11, YOLOv8, and Mask R-CNN for Plastic Waste Detection"**

## Repository Structure

```
dataset/
models/
training/
results/
weights/
configs/
```

## Environment

- Python 3.12
- PyTorch
- Ultralytics YOLO
- CUDA
- NVIDIA Quadro P4000 GPU

## Models

- YOLOv11
- YOLOv8
- Mask R-CNN

## Training

- Random Seed: 42
- Image Size: 640 × 640
- Batch Size: 16
- Epochs: 50

## Dataset

The dataset consists of annotated plastic waste images collected from multiple publicly available sources and manually annotated using Roboflow.
